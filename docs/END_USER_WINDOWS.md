# NC Invoice Manager — Windows lietotāja instrukcija

## Ko darīt
1. Lejupielādē `NCInvoiceManager-windows.zip`.
2. Izpako jebkurā mapē.
3. Palaid `NCInvoiceManager.exe` (double click).
4. Sagaidi ziņu CMD logā, ka serveris gatavs.
5. Pārlūks atvērsies automātiski uz `http://127.0.0.1:8000`.

## Ikdienas lietošana
- Lai startētu aplikāciju: vienkārši atkal palaid `NCInvoiceManager.exe`.
- Lai apturētu aplikāciju: aizver CMD logu vai nospied `CTRL+C`.

## Dati
- Datu fails: `nc_invoice_manager.db` (blakus EXE).
- Nepazaudē šo failu, ja pārnes aplikāciju uz citu datoru.
